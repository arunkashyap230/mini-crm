# Mini CRM System

A simple CRM application built with React for managing customer leads.

## Features
- Fetch leads from API (JSONPlaceholder)
- Add, Edit, Delete leads
- Search by name or company
- Filter by lead status
- Data persistence with localStorage

## Tech Stack
- React (Functional Components)
- Axios (API calls)
- CSS3
- JavaScript (ES6+)

## Setup
```bash
npm install
npm run dev
```
